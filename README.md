# PM Copilot

Интеллектуальный copilot для senior PM в финтехе. Сокращает время от инсайта до готовых бизнес-требований и ПРД — с недель до дней.

Работает внутри Claude Code — помогает на каждом этапе: от декомпозиции целей до валидированного ПРД и коммуникаций.
Не набор разрозненных скиллов, а stateful copilot — собирает, сохраняет и обновляет рабочий контекст между сессиями, помогая PM быстрее принимать осознанные решения.

**Для кого:** Senior PM, ведущий 3-5 инициатив параллельно с собственной командой разработки и взаимодействием со смежными командами.

**Что умеет:**
- Декомпозировать цели на эпики, эпики на user story и задачи
- Из цели → к обоснованным эпикам и задачам
- Короткий путь: из гипотезы → к готовому ПРД с метриками, рисками и ганардами
- Из ПРД → к коммуникациям для команды и C-level
- Локально сохраняет контекст между сессиями — не переспрашивает одно и то же
- Поддерживает работу с несколькими инициативами параллельно

**Версия:** v6.0 (Goal-First Architecture)

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/dewbbk/pm-copilot/main/install.sh | bash
```

Скрипт скачает скиллы и установит их в `~/.claude/skills/`.

---

## Обновление

Та же команда — скрипт сам подтянет изменения:

```bash
curl -fsSL https://raw.githubusercontent.com/dewbbk/pm-copilot/main/install.sh | bash
```
---

## Запуск

В Claude Code напиши:

```
/pm-copilot
```

При первом запуске нужно пройти онбординг. Пайп соберет и сохранит контексты (3-4 вопроса о домене, продукте, ключевых метриках и целях)
