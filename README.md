# PM Copilot

Работает внутри Claude Code — помогает на каждом этапе: от декомпозиции целей до валидированного ПРД и коммуникаций.
Помогает принимать правильные решения

**Что умеет:**
- Декомпозировать цели на вехи, вехи на user story и задачи
- Из инсайта или проблемы → к обоснованной цели и гипотезам
- Из гипотез → к задачам и к готовому ПРД с метриками, рисками и корректирующими метриками
- Из ПРД → к коммуникациям для команды и C-level
- Локально сохраняет контекст между сессиями — не переспрашивает одно и то же
- Поддерживает работу с несколькими инициативами параллельно

**Версия:** v4.20 (Sprint 28)

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/dewbbk/pm-copilot/main/install.sh | bash
```

Скрипт скачает скиллы и установит их в `~/.claude/skills/`.

---

## Обновление

Та же команда — скрипт сам подтянет изменения:

```bash
curl -fsSL https://raw.githubusercontent.com/dewbbk/pm-copilot/main/install.sh | bash
```
---

## Запуск

В Claude Code напиши:

```
/pm-copilot
```

При первом запуске нужно пройти онбординг. Пайп соберет и сохранит контексты (3-4 вопроса о домене, продукте, ключевых метриках и целях)
