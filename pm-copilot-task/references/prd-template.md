# Шаблон карточки ПРД

> **Назначение**: Единый формат ПРД для банковского продукта, который обеспечивает полноту проработки и связь с царь-метриками.

## Как использовать

Заполняй этот шаблон на основе ответов PM, собранных в пути 3 (task-path.md). Не придумывай то, что PM не сказал — оставляй `[УТОЧНИТЬ]`.

## Правила заполнения

1. **Не оставляй пустых секций** — если данных нет, пиши `[УТОЧНИТЬ У PM]` или `[ЗАПРОСИТЬ У АНАЛИТИКА]`
2. **Метрики — только подтверждённые** — не придумывай цифры, лучше оставь пустыми
3. **Стейкхолдеры — реальных людей** — не «команда разработки», а конкретные имена
4. **Цепочка влияния обязательна** — без неё ПРД не валиден
5. **Ганарды обязательны** — без них ПРД не валиден
6. **Секция 8 — красная линия** — если связь с царь-метрикой «Слабое», задача под вопросом
